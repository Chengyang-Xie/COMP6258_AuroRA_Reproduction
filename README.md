# COMP6258 AuroRA Reproduction

This repository contains a constrained reproduction of:

**AuroRA: Breaking Low-Rank Bottleneck of LoRA with Nonlinear Mapping**

## Important note
 
LoRA and AuroRA were implemented from scratch in PyTorch.

## Reproduction scope

- Model: roberta-base
- Tasks: MRPC, RTE, CoLA from GLUE
- Methods:
  - LoRA rank 2
  - LoRA rank 8
  - AuroRA rank 2
- Ablation:
  - Full AuroRA
  - AuroRA without fixed tanh nonlinearity
  - AuroRA without spline nonlinearity

## Files

- `AuroRA_reproduction.ipynb`: complete notebook
- `results/results_raw.csv`: raw experimental results
- `results/results_summary.csv`: aggregated results
- `figures/`: figures generated by our project
- `report/report.pdf`: final report

## How to run

1. Open `AuroRA_reproduction.ipynb` in Google Colab.
2. Select GPU runtime.
3. Run all cells in order.
4. Results will be saved to Google Drive.
